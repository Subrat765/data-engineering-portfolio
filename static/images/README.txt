# Images Directory

This directory contains image assets for the Data Engineering Portfolio website.

## Usage
- Add portfolio project screenshots
- Store hero background images
- Include profile photos or avatars
- Add any other visual assets needed for the website

## Supported Formats
- PNG (recommended for screenshots and images with transparency)
- JPG/JPEG (recommended for photos and large images)
- SVG (recommended for icons and logos)
- GIF (for animated content if needed)

## File Naming Convention
- Use descriptive names
- Use lowercase letters
- Separate words with hyphens (-)
- Examples: hero-background.jpg, project-screenshot-1.png, profile-photo.jpg

## Git Tracking
This README.txt file ensures the images directory is tracked by Git even when empty.
